# DSDPlus-VIPER-BAT-Files
Collection of .bat files to use with DSDPlus on the VIPER system. Each are labeled by site name. Data was gathered from open sources. Use with due regard to public safety opearations.

Extract the .bat files to your DSDPlus directory in Windows. Optionally, you can place the DSDPlus VIPER Launcher.py file in the same directory and get a GUI to use based on the county you're in. You may need to modify the control channels in the .bat file based on your usage. You can also change the control channel in the python app if you know where to look.
